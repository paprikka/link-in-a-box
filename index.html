Link in a box. How to share Signal links on Twitter

Twitter is blocking links to Signal. Use this tool to circumvent that.

Package link into a data URI

Turn a link into a data URI. A data URI is a link that contains the content of the link in the link itself. In our case this will be a tiny page with a link poining to Signal. 


            
          


          

            
            
          

        

      


      

        
Add a space to the URL

        

          Replace
          https://signal.me/#eu/P01wpU[...]EttS85C
          with
          https://signal .me/#eu/P01wpU[...]EttS85C
        

        

          It works. It might be confusing to some users if they don't spot the
          space in the link.
