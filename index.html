Link in a box. How to share Signal links on Twitter

Twitter is blocking links to Signal. Use this tool to circumvent that. 


            
          

          

            
            
The link should copy-pasted

            
          

        


        
How it works

        

          Turn a link into a data URI. A data URI is a link that contains the
          content of the link in the link itself. In our case this will be a
          tiny page with a link poining to Signal.
          

          This is inspired by an older project of mine:
          Laconic.
        

      

      
      
Other things you can try

      

        
Add a space to the URL

        

          Replace
          https://signal.me/#eu/P01wpU[...]EttS85C
          with
          https://signal .me/#eu/P01wpU[...]EttS85C
        

        

          It works. Keep in mind that it might be hard for some people to spot
          the space in a link. The main reason I'm using the box is that it's
          explicit.
        

      

      

        
Share as a QR code

        
Turn the link into a QR code using some of the free tools online.

        

          It works. But, you'll need a QR code reader to follow the link.
